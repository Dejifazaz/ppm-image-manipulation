# PPM Image Manipulation

A simple C++ application to load, save and manipulate PPM (P6) image files.

## Features
- Load and save PPM files
- Filters: red, green, blue, greyscale
- Flip horizontal and vertical
- Advanced: rotate 90°, mirror, blur, crop, 4-bit, and more

## Build
Use the SFML GUI project from Moodle and replace the Image.h and Image.cpp files with the ones in this repo.
